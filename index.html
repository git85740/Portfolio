THE MAZE

Unity Horror Game

A Unity based maze exploring game with a horror theme. The goal of the game is to find the key and escape through the locked door whilst being hunted by enemies. Makes use of an AI State Machine to allow the AI to have different behaviors based on the player's actions.

Used Languages: C#

Used Tools: Blender, GIMP, Audacity

THE HOUSE IN THE WOODS

JavaScript Adventure Game

A JavaScript based text adventure game. Your goal is to explore a seemingly abandoned mansion to uncover the whereabouts of it's missing owner. Makes use of JS based states to check the player's inventory for items and actions the player has taken.

Used Languages: HTML, CSS, JS

Used Tools: Audacity

Website

DUNGEON CRAWL

Unity Retro Action Game

A Unity based PS1 style action game. The game is a proof of concept designed in only 3 days for a school exposition. The goal of the game is to get to the end of the dungeon whilst fighting skeletons and avoiding falling into any pits. Makes use of multiple filters and self-made textures to give the game a unique Playstation 1 aesthetic. Further achieves this through the low poly models also made by myself.

Used Languages: C#

Used Tools: Blender, GIMP, Audacity
